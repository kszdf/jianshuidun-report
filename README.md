## 🎯 产品定位

- **不是财务软件**：不做账，只用账
- **不是通用BI**：建筑行业开箱即用，零配置起步
- **是老板的驾驶舱**：一屏掌握钱、项目、应收、利润

**核心理念：不增加财务负担，财务只做一次账。**

## ✨ 核心功能

### 🎯 v0.2.0 核心三剑客

#### 1. 项目独立核算看板 ⭐ 老板最刚需
- 每个项目一本账，赚不赚钱一眼清
- 三种收入确认模式自由切换：开票口径 / 结算口径 / 完工百分比
- 项目列表：合同金额、已开票、已收款、已发生成本、毛利、毛利率、进度
- 单项目详情：收入分析、成本明细（人材机分包）、毛利分析、资金情况、发票情况
- 一个公司N个项目独立核算，支持合伙人各管各的项目

#### 2. 挂靠税费精算计算器 🔥 杀手级传播功能
- 5个变量全部手动可调：合同金额、适用税率、挂靠费比例、增值税税负率、所得税税负率
- 一键算出20+项结果：不含税金额、销项税、增值税、附加税、所得税、印花税、管理费、合计扣款、进项税需求、成本票需求、账面利润、分红个税...
- 支持「先扣后返」模式：按无票最大情况预扣，再根据进项票和成本票返还
- 3个预设场景一键切换：一般计税大包 / 简易计税 / 劳务分包
- 综合费率自动计算，老板秒懂

#### 3. 税负率与缺票分析 ⚠️ 风险预警
- 增值税税负率 + 所得税税负率实时监控
- 与行业参考值对比，偏高/偏低自动预警
- 进项税缺口测算：理论应取 vs 实际取得
- 成本票缺口分析：分材料/人工/机械/分包明细
- 分项目税负对比表，哪个项目有问题一目了然

### 📚 人材机占比参考库
内置7类工程行业参考值：
| 工程类型 | 人工费 | 材料费 | 机械费 |
|---------|-------|-------|-------|
| 房建工程 | 30% | 60% | 10% |
| 市政工程 | 20% | 55% | 25% |
| 装修工程 | 40% | 55% | 5% |
| 劳务工程 | 95% | 3% | 2% |
| 安装工程 | 50% | 40% | 10% |
| 水利工程 | 10% | 40% | 50% |
| 园林工程 | 40% | 50% | 10% |

支持项目实际值与行业参考值对比，自动分析异常项并给出建议。

### 📊 经营驾驶舱（v0.1.0）
6大核心指标 + 3张可视化图表 + 智能预警，老板一眼全局

| 指标 | 说明 |
|------|------|
| 本月收入 | 当期营业收入 |
| 本月成本 | 当期营业成本 |
| 本月毛利 | 收入-成本 |
| 毛利率 | 毛利/收入 |
| 应收账款 | 应收余额 |
| 货币资金 | 银行+现金余额 |

### 💰 项目利润表
按项目维度分析收入、成本、毛利、毛利率，支持钻取明细

### 💵 资金日报
多银行账户余额一览 + 收支流水明细

### 📋 应收账龄分析
30/60/90/180天分段，超期预警，辅助催收

### 🏗️ 项目独立核算看板 <span style="color:#c9a24a">v0.2新增</span>
N个项目独立核算，一屏掌握所有项目经营状况

- **项目列表**：收入/成本/毛利/毛利率/资金/发票全景对比
- **三种收入确认模式**：开票口径 / 结算口径 / 完工百分比，一键切换
- **单项目详情**：成本结构明细、资金收支、发票进销项、工程结算对冲
- **人材机占比分析**：项目实际 vs 行业参考值对比
- **税负率 & 缺票分析**：增值税/所得税税负率预警，进项缺口测算

### 🧮 挂靠税费精算计算器 <span style="color:#c9a24a">v0.2新增</span>
5个输入变量，20+项计算结果，挂靠业务一键精算

- **标准模式**：合同金额/税率/挂靠费/增值税负/所得税负 → 综合费率+到手金额
- **先扣后返模式**：预扣全额税费，提供进项票后返还，精准管控资金
- **3个预设场景**：房建9% / 市政9% / 装修3%，一键加载演示
- **缺票测算**：自动计算需要多少进项票才能控制税负率

### 📚 人材机参考库 <span style="color:#c9a24a">v0.2新增</span>
7类建筑工程人材机占比内置参考值

- 房屋建筑工程、市政公用工程、装饰装修工程
- 机电安装工程、公路工程、水利水电工程、钢结构工程

### 📥 Excel智能导入
- 支持所有主流财务软件（用友/金蝶/速达/管家婆等）导出的Excel
- 自动识别表头、自动匹配科目
- 支持科目余额表 + 凭证序时账两种导入方式

### 🔗 科目自动映射
内置49个建筑行业标准科目，智能关键词匹配，第一次映射后续自动沿用

## 🚀 快速启动

### 方式一：最快（单文件版，30秒跑起来）

```bash
# 1. 克隆代码
git clone https://github.com/kszdf/jianshuidun-report.git
cd jianzao-report/backend

# 2. 安装依赖
pip install -r requirements.txt

# 3. 初始化数据
python init_data.py

# 4. 启动服务
uvicorn main:app --reload --port 8000
```

打开浏览器访问：**http://localhost:8000/**

### 方式二：前端工程化开发

```bash
# 后端同上启动

# 前端
cd frontend-vite
npm install
npm run dev
```

访问：**http://localhost:5173**

## 🏢 行业背景

本产品由**慧根堂财税联盟**联合开发，基于25年建筑行业财税实战经验，深度理解建筑企业的管理痛点和财务人员的工作习惯。

**为什么做这个产品？**
- 建筑企业老板看不懂财务账，财务出管理报表要加班
- 市面上的通用BI太贵、太复杂，建筑行业还得定制
- 财务软件自带的报表都是会计视角，不是老板视角
- 换财务软件成本太高，不现实

**我们的解法：不换软件、不加工作量，导出Excel就出报表。**
