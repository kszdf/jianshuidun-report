## v0.2.0 — 项目核算与挂靠税费（2025年7月）

> 大版本升级：从"财务报表"升级到"经营管理"，新增项目独立核算看板和挂靠税费精算两大核心模块

### ⭐ 核心亮点

1. **项目独立核算** — 从"公司整体"深入到"单个项目"，每个项目独立核算收入、成本、毛利、资金、发票
2. **三种收入确认模式** — 开票口径 / 结算口径 / 完工百分比，满足不同管理需求
3. **挂靠税费精算计算器** — 5个输入变量，20+项计算结果，支持先扣后返模式
4. **人材机参考库** — 7类建筑工程内置参考值，项目实际与行业对比
5. **工程结算科目修正** — 5402工程结算从收入类改为成本类备抵科目，修正专业硬伤

### ✨ 新功能

#### 1. 项目独立核算看板
- 项目列表：收入/成本/毛利/毛利率/资金/发票全景对比
- 三种收入确认模式：开票口径 / 结算口径 / 完工百分比（手动比例）
- 单项目详情页：
  - 成本明细：人工/材料/机械/分包/其他费用分类展示
  - 资金情况：已收款/已付款/资金结余
  - 发票管理：销项发票/进项发票/进销项对比
  - 工程结算：工程施工 vs 工程结算对冲分析
- 人材机占比分析：项目实际 vs 行业参考值对比
- 税负率 & 缺票分析：增值税/所得税税负率预警，进项缺口测算

#### 2. 挂靠税费精算计算器
- **标准模式**：
  - 输入：合同金额、适用税率、挂靠费比例、增值税税负率、所得税税负率
  - 输出：不含税金额、销项税额、各项税费明细、综合费率、实际到手金额
  - 缺票测算：需要多少进项票才能控制税负率
- **先扣后返模式**：
  - 预扣全额税费（增值税+附加+所得税+印花税+挂靠费）
  - 提供进项票后按比例返还
  - 精准管控挂靠业务资金风险
- **3个预设场景**：
  - 房建工程（100万，9%，2%挂靠，2.5%增值税负，1.5%所得税负）
  - 市政工程（500万，9%，1.5%挂靠，2%增值税负，1%所得税负）
  - 装修工程（50万，3%，3%挂靠，1%增值税负，0.5%所得税负）

#### 3. 人材机参考库
- 独立页面展示7类建筑工程人材机占比参考值：
  - 房屋建筑工程：人工30% / 材料60% / 机械10%
  - 市政公用工程：人工20% / 材料55% / 机械25%
  - 装饰装修工程：人工40% / 材料55% / 机械5%
  - 劳务工程：人工95% / 材料3% / 机械2%
  - 机电安装工程：人工50% / 材料40% / 机械10%
  - 水利水电工程：人工10% / 材料40% / 机械50%
  - 园林工程：人工40% / 材料50% / 机械10%
- 工程类型卡片切换 + 饼图可视化 + 全类型对比表格
- 项目详情页支持「项目实际 vs 行业参考」对比分析

### 🔧 技术变更

#### 后端
- 新增模型 `business_ext.py`：
  - `BizProjectAccounting` — 项目核算汇总表
  - `BizCashflowForecast` — 现金流预测表（预留）
  - `BizLmrReference` — 人材机参考库
  - `BizInvoiceSummary` — 发票汇总表
  - `BizProjectBudget` — 项目预算表（预留）
- 新增服务 `tax_calc_service.py`：
  - `calculate_affiliated_tax()` — 挂靠税费计算
  - `calculate_pre_deduct_return()` — 先扣后返计算
  - `get_lmr_reference()` — 获取人材机参考库
- 新增服务 `project_accounting_service.py`：
  - `get_board()` — 项目看板列表
  - `get_project_detail()` — 项目详情
  - 支持三种收入模式切换
  - 成本匹配与工程结算对冲
- 新增API路由 `tax_calc.py`：
  - `POST /api/tax/affiliated-tax` — 挂靠税费计算
  - `POST /api/tax/pre-deduct-return` — 先扣后返计算
  - `GET /api/tax/lmr-reference` — 人材机参考库
- 新增API路由 `project_accounting.py`：
  - `GET /api/project-accounting/board` — 项目看板列表
  - `GET /api/project-accounting/detail` — 项目详情

#### 前端
- 新增页面 `ProjectBoard.vue` — 项目核算看板列表
- 新增页面 `ProjectBoardDetail.vue` — 项目详情（4个Tab）
- 新增页面 `AffiliatedTaxCalc.vue` — 挂靠税费计算器
- 路由和菜单已同步更新
