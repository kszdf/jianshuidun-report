# 建税盾 v0.2.0 部署指南

## 📋 前置准备

- 腾讯云服务器：Ubuntu 22.04 LTS，2核2G以上
- 域名：hgtcs.com（已备案）
- DNS解析：
  - report.hgtcs.com → 服务器IP
  - sjk.hgtcs.com → 服务器IP

## 🚀 一键部署

### 步骤1：上传代码包

将 `jianshuidun_v0.2.0.tar.gz` 上传到服务器 `/root/` 目录

```bash
# 本地执行（或用WinSCP等工具上传）
scp jianshuidun_v0.2.0.tar.gz root@服务器IP:/root/
```

### 步骤2：登录服务器

```bash
ssh root@服务器IP
```

### 步骤3：上传并解压部署脚本

```bash
cd /root
# 解压代码包
tar xzf jianshuidun_v0.2.0.tar.gz

# 进入目录，复制deploy.sh到root
cp jianzao-report/deploy.sh /root/
chmod +x deploy.sh
```

### 步骤4：执行部署

```bash
./deploy.sh
```

部署过程约5-10分钟，完成后会显示访问地址。

## 🔒 HTTPS证书

部署完成后，确认DNS已解析，执行以下命令申请免费SSL证书：

```bash
certbot --nginx -d report.hgtcs.com -d sjk.hgtcs.com
```

按照提示输入邮箱，选择自动重定向（选2）即可。

## 🔧 常用命令

### 后端服务

```bash
# 查看状态
systemctl status jianshuidun

# 重启
systemctl restart jianshuidun

# 查看日志
journalctl -u jianshuidun -f
```

### Nginx

```bash
# 测试配置
nginx -t

# 重载
nginx -s reload

# 查看日志
tail -f /var/log/nginx/access.log
tail -f /var/log/nginx/error.log
```

## 🔑 飞书多维表对接（可选）

如需启用税检康线索自动写入飞书多维表，配置环境变量：

```bash
# 编辑后端配置
nano /etc/systemd/system/jianshuidun.service

# 在 [Service] 段添加以下环境变量
Environment="FEISHU_APP_ID=你的appId"
Environment="FEISHU_APP_SECRET=你的appSecret"
Environment="FEISHU_BITABLE_APP_TOKEN=你的base_token"
Environment="FEISHU_BITABLE_TABLE_ID=你的table_id"

# 重启服务
systemctl daemon-reload
systemctl restart jianshuidun
```

## ❓ 常见问题

### 访问502 Bad Gateway
后端服务未启动，检查：
```bash
systemctl status jianshuidun
```

### 前端页面空白
检查前端是否构建成功：
```bash
ls /opt/jianshuidun/frontend-vite/dist/
```

### 端口被占用
```bash
# 查看8000端口占用
lsof -i :8000
```
