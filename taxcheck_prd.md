# 税检康 · 财税风险智能检测 产品文档 v1.0

## 一、产品定位
慧根堂旗下轻量级财税风险检测工具，作为官网核心流量入口，通过"快速测风险"的钩子获取企业老板/财务的联系方式，为代账、税务规划、稽查应对等高客单产品引流。

## 二、核心闭环（必须跑通）

```
用户进入 → 选择行业 → 答题（20题）→ 实时算分 → 查看基础报告
    → 留资领完整报告 → 飞书多维表入库 → 飞书通知张老师 → 后续跟进
```

| 层 | 技术选型 | 说明 |

## 四、题库设计（20题，5大维度）

### 维度1：发票与收入风险（4题）
1. 近一年是否存在大额咨询费/服务费发票，且无实质性业务支撑？
2. 是否存在大额现金交易（单笔5万以上）且无完整凭证链？
3. 是否存在"进销项严重背离"情况（如进项全是建材，销项全是服务）？
4. 是否存在长期挂账的预收账款超过12个月未结转收入？

### 维度2：成本费用风险（4题）
5. 是否存在大量无票支出或白条入账？
6. 业务招待费、福利费、广告费是否远超行业常规比例？
7. 工资薪金是否全部通过公户发放并如实申报个税？
8. 是否存在股东个人消费在公司报销的情况？

### 维度3：资金与往来风险（4题）
9. 其他应收款/其他应付款余额占资产总额比例是否超过30%？
10. 是否存在股东或关联方长期大额占用公司资金（超过1年未归还）？
11. 是否频繁发生公转私（单笔10万以上）且用途模糊？
12. 是否存在多个银行账户间无实质业务的频繁资金划转？

### 维度4：税务申报风险（4题）
13. 增值税税负率是否明显低于同地区同行业平均水平（偏差超30%）？
14. 企业所得税申报收入与增值税申报收入是否存在较大差异？
15. 是否存在连续6个月以上零申报或负申报？
16. 个税申报工资总额与企业所得税工资支出是否匹配？

### 维度5：金税四期与合规意识（4题）
17. 公司及法人/股东近3年是否被税务机关检查或预警过？
18. 是否存在社保缴纳人数与个税申报人数差异较大的情况？
19. 近一年是否有新增大额关联方交易且定价不公允？
20. 是否有专业财税人员定期做税务风险自查？

## 五、评分规则

### 答题方式
- 每题3个选项：A=是（高风险，5分）/ B=部分存在（中风险，3分）/ C=否（无风险，0分）
- 满分100分（20题×5分），分数越高风险越大

### 风险等级
| 分数区间 | 等级 | 结果页主色 | 说明 |
|---------|------|-----------|------|
| 0-20分 | 低风险（绿灯） | 绿色 | 整体财税管理较规范 |
| 21-45分 | 中风险（黄灯） | 橙色 | 存在若干风险点，建议关注 |
| 46-70分 | 高风险（红灯） | 红色 | 风险较突出，建议尽快排查 |
| 71-100分 | 极高风险（警报） | 深红 | 存在重大税务风险，需立即处理 |

### 基础报告（留资前可见）
- 总分 + 风险等级
- 雷达图（5个维度得分）
- TOP 3 风险点简述（每题1句话点出问题）
- "获取完整报告和一对一解读"留资按钮

### 完整报告（留资后通过微信/邮箱发送）
- 全部20题的详细分析
- 每个风险点的具体说明 + 改进建议
- 行业对标参考
- 慧根堂联系方式 + 免费咨询入口

## 六、留资字段
- 企业名称（选填？→ 必填，增加真实性）
- 姓名（必填）
- 手机号（必填，手机号验证→暂不做，降低门槛）
- 行业（从答题时已选，自动带入）
- 检测日期（自动）
- 总分/等级（自动）

> 注：留资门槛要低——字段越少越容易填。MVP版只留：姓名 + 手机号 + 企业名称。三个字段，不多不少。

## 八、页面流程

1. **首页**（首屏大图 + 产品介绍 + 开始检测按钮）
2. **行业选择**（7个行业图标任选）
3. **答题页**（20题逐题作答，进度条）
4. **计算中**（1-2秒加载动画）
5. **基础报告页**（分数 + 雷达图 + TOP3风险 + 留资领完整报告）
6. **留资表单**（3个字段 + 提交）
7. **提交成功页**（提示"顾问将在24小时内联系您" + 公众号二维码 + 电话）

## 九、财税专业严谨性保障

1. 所有风险点描述均基于现行有效税法（增值税法、企业所得税法、个人所得税法、税收征管法）
2. 不承诺具体结果、不打包票、不涉及"偷税漏税指导"
3. 报告底部统一加免责声明："本检测结果仅供参考，不构成税务建议，具体税务处理请咨询专业人士"
4. 金税四期相关表述基于公开信息，不夸大、不恐吓
